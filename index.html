Welcome to Open* for ICTD

Information and Communication Technology for Development ("ICTD"), also known as Information and Communications Technologies for Development or "ICT4D", refers to the use of Information and Communication Technologies (ICTs) in the fields of socioeconomic development, international development and human rights. The theory behind this is that more and better information and communication furthers the development of a society.[1]

Open* for ICTD is a practical, open system framework supporting integrated [integrative?] approaches to challenges faced by ICTD. It is not the whole picture, but provides a base for building upon.

Learn More

The Open* for ICTD framework embraces Openness and these more specific concepts:

  • Open Standards
  • Open Data
  • Open Access
  • Open Source

As indicated by the asterisk ("Open*") in our framework's title, this list will continue to grow. It starts with these four (4). Critically, these four concepts are all ready to be applied in the form of content, practices and tools that are widely available.

Toolkit

For our next release, Open* for ICTD will assembling a toolkit of technologies and processes to guide initial implementation of this framework. This toolkit will include elements from the list below covering the range of concepts noted above:

  • W3C Web Standards
  • ORCID
  • Figshare
  • arXiv
  • GitHub
