Welcome to Open* for ICTD

Open* for ICTD is an Open System framework supporting new integrated approaches to serious challenges faced in Information and Communication Technology for Development.

Open* for ICTD diagram here...

Learn More

The Open* for ICTD framework embraces the following general concepts, all of which are backed by concrete instances in global use:

  • Open Standards
  • Open Data
  • Open Access
  • Open Source
  • Open APIs

As indicated by the asterisk ("*") in our framework's title, this list will continue to grow

Toolkit

We're assembling a toolkit of technologies and processes to guide initial implementation of this framework.

The beta version of our Open* for ICTD toolkit provides examples covering the range of concepts noted above:

  • W3C Web Standards
  • ORCID
  • Figshare
  • arXiv
  • GitHub
